# Bokorm support

Statisk norsk supportside for Bokorm.

## GitHub Pages

Anbefalt oppsett:

1. Opprett et offentlig GitHub-repo med navnet `bokorm-support`.
2. Push innholdet i denne mappen til repoet.
3. Gå til `Settings` -> `Pages`.
4. Velg `Deploy from a branch`.
5. Velg `main` og `/ (root)`.

Support-URL-en blir normalt:

```text
https://haugesander.github.io/bokorm-support/
```
